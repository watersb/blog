Longer-Term

Alternative FrontEnds might be of interest. I know that when I get into long-form articles, like this one has turned into, that I start adding structure to the document, an outline form. It might be possible to continue with this text editor and some macros to support the structure, or it might be of interest to use a different front-end like OmniOutliner or MicrosoftWord.

I’ve gone down this road over and over again, and I’m put off by a lack of control over the resulting output format. But maybe I don’t care, as long as I have a way to go from the format required by the front-end (probably an XML schema of some sort) and properly-formatted text. Hmm. Seems like a lot of work, but I do consider the notion of learning more about my design by listening to sophisticated front-ends.

Because frankly re-ordering an outline hierarchy is a lousy task to try with a non-trivial document — getting all of the section’s children in there properly is easy to miss with a straight text editor. I note that for TextEdit, an appropriate language grammar that supports folding would probably do just fine; note that computer source code is an example of a very non-trivial structured text document.

Would you like to hear my tale of woe regarding Windows installation? No? Then go elsewhere; I’m going to rant a bit.

Over the past four days, I’ve tried to install Windows on a couple of old computers. I’ve been volunteering to putter around with cast-off equipment, to refurbish it enough to make it worthy of donation to a person or charity. Since I’ve become disabled, this sort of thing is something that I can do with little or no external pressure, and it helps me get through the days when I need something to take my mind off of physical discomfort.

So I suppose that when I’m working with Windows, I’m not in the best of mental and physical states, anyway.

But about old stuff: I cannot get old versions of Windows to install. It’s quite remarkable, actually. Microsoft does an amazing job in supporting their developers and customers - old programs will generally run fine on new versions of Windows. But for a home-hobbyist like me, it’s practically impossible to re-install the operating system on an old computer. Which renders these things worthless.

Windows is hard to maintain, and most people who use it are not computer professionals. The typical issue is a machine that’s bogged down with so much crapware that it spends all of its time swapping, like swimmers who can’t catch their breaths. Simply adding more RAM is a great way to extend the life of an old computer.

But the best thing for old computers is a wipe, a re-install of the operating system, and a simplification of the user workflow. They either need a newer computer, or they need to keep things simple. Windows doesn’t scale very well past a certain point, not on older hardware. I’m talking about more than five years old here. Time for a new machine.

But it is galling that a wonderful computer from just a while ago is thrown away — it doesn’t take much to host a web browser or children’s educational software. Or it shouldn’t…

In any event, all this Microsoft Genuine Advantage stuff prevents me from installing the dozen or so disks of Windows OS that I’ve managed to accrete. It may be tied to a vendor, a particular computer, or obsolete.

And don’t get me going on “upgrade” versions. I’d love to accommodate this, but it doesn’t work. I can install Windows 98 and then try to run the XP updater, no luck. It’s all Windows, but they don’t talk to one another.

With Linux, which is open-source and free, there are compatibility issues, sure, but Linux doesn’t ask you for an activation key. Neither does Mac OS X, as Apple is interested in all Macs running the same (or nearly the same) version of the OS, in order to drive workstation sales. Apple makes money off the hardware, not the software. Duh.

Microsoft can’t do that, can’t afford to give it away for free, must (by accounting rules, never mind the business proposition) charge something for their software. They make it available for low prices to students and researchers. But there’s still all the hoops of upgrades, activation, and product keys. Locked out.

Let me clarify something: I’m ranting about this because I’d like to clean up computers that have valid Windows COA stickers on them, or I’d like to be able to install the version of Windows that I personally own onto a VMWare machine. These are two different scenarios, but they are both blocked (or made prohibitively difficult) by the product key hurdles.
